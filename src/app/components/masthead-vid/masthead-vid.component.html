About this project

This project is to study Angular and TailwindCSS with modern UI/UX ideas. The components are mainly inspired by @Hyperlexed on YouTube. The goal was to translate his reverse engineered components from plain HTML/CSS/JS to Angular and TailwindCSS. Scrolling through Awwwards' Site Of The Day was also inspirational.

Used Tech:

  • Angular
  • TailwindCSS
  • ThreeJS
  • GSAP
  • Strapi
